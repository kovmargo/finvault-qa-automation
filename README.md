# FinVault QA Automation

End-to-end QA automation project covering a custom-built fintech REST API — designed specifically to practice and demonstrate Senior-level QA Automation skills.

## Stack

| Layer | Tools |
|---|---|
| Backend API tests | Python, pytest, requests, pydantic, OOP |
| Auth testing | JWT, OAuth2, refresh tokens, role-based access |
| Database | SQL, PostgreSQL |
| Frontend UI tests | TypeScript, Playwright |
| Code quality | ESLint, mypy |
| CI/CD | GitHub Actions, GitLab CI |
| Infrastructure | Docker, AWS basics |
